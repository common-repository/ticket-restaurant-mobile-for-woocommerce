Payment Gateway that allows customers to pay with Ticket Restaurant®

== Description ==

Payment gateway for WooCommerce that allows you to receive payments with Ticket Restaurant®

= Features: =

Ticket Restaurant is a benefit solution that companies provide to their employees. It enables both tax and social advantages for daily meals. The prepaid card along with My Ticket is the most practical electronic solution to pay your meals on a daily basis. Install the official plugin so that your business can accept mobile payments from Ticket Restaurant users. Please note that you must have an active agreement with Ticket Serviços. Access our [merchants portal](https://aderentes.ticket.pt) at https://aderentes.ticket.pt. Technical support is provided by Paycritical.

This plugin allows you to receive payments from [Ticket Restaurant](https://www.ticket.pt) users on your system through the checkout process while processing transactions through a 3rd party Gateway in the background. This allows you to provide a smoother, more complete checkout process to the customer. Ticket Restaurant Portugal is a Portuguese company that issues meal vouchers. This plugin is provided my Paycritical Soluções de Pagamento LDA.

When you install this Plugin, end users are requested to introduce their phone number, so that payment requests are received on their Ticket Restaurant mobile app, known as “My Ticket”. This means that phone number information is exchanged and transmitted through internet data services. [Terms and Conditions](https://ticket.pt/pdfs/TR-Cartao-CGU_5ed6317577b310-10796946.pdf)

== Installation ==

* Use the included automatic install feature on your WordPress admin panel and search for "Ticket Restaurant for WooCommerce".
* Go to WooCoomerce > Payment and activate "Ticket Restaurante Mobile" gateway
